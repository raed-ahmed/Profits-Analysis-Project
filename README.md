# 📊 Profits Analysis Project 

## 📌 Overview

The **Profits Analysis Dashboard** provides a comprehensive view of business performance, focusing on profits, revenue, and key financial metrics. It enables users to analyze trends, identify top-performing areas, and make data-driven decisions.

## 🚀 Features

* 📦 **Orders Tracking**

  * Total number of orders: **5,009**

* 📊 **Sales & Profit Metrics**

  * Total Sales: **$2.30M**
  * Total Profits: **$286.01K**
  * Profit Margin: **12.02%**
  * COGS: **$2.01M**

* 📈 **Yearly Analysis**

  * Best Year: **2023**
  * Contribution: **33% of total profits**

* 🧩 **Category Insights**

  * Top Category: **Technology**
  * Contribution: **51% of total profits**

* 📅 **Monthly Trends**

  * Highest Profit Months: **December, September, November**
  * Contribution: **41% of total profits**

* 🗓️ **Week Type Analysis**

  * Weekdays generate **70%** of total profits

* 🌍 **Regional Performance**

  * Top Region in Profits: **West (34%)**
  * Top Region in Revenue: **West (38%)**

* 🏆 **Top Customers**

  * Displays top 10 customers contributing to profits
  * Top contributor share: **17.5%**
    
## 🎛️ Dashboard Controls

* 🔄 **Clear Filter**: Reset all applied filters
* 🏠 **Back to Home**: Navigate to main page
* 📅 **Year Filters**:

  * 2020
  * 2021
  * 2022
  * 2023

## 🛠️ Technologies Used

This dashboard was built using :

* **Microsoft Excel**
* **Power Query** (Data Cleaning & Transformation)
* **Power Pivot** (Data Modeling & Relationships)
* **Dashboard Design** (Interactive UI & Visualization)
